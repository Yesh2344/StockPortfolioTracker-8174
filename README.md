## Introduction
The Stock Portfolio Tracker is a Python application designed to help investors manage their stock portfolios. It provides a simple and intuitive way to track stock prices, calculate portfolio value, and generate reports.

## Installation
To install the Stock Portfolio Tracker, follow these steps:
1. Clone the repository using `git clone https://github.com/your-username/StockPortfolioTracker.git`
2. Navigate to the project directory using `cd StockPortfolioTracker`
3. Install the dependencies using `pip install -r requirements.txt`
4. Create a new file named `.env` and add your API keys and configuration settings
